✧ variable

variable

⤞ variable is like a container where we can store the value

⤞ Types Of Declaring variables

  • 1. let
  • 2. var
  • 3. const

⤞ let

Variables defined with let can not be redeclare or let can be reinitialize.

⤞ Syntax 
                    let x = "John Doe";
                    let y = 0;

⤞ var

Variables defined with var can be redeclare or var can be reinitialize.

⤞ Syntax 
                    let x = "John Doe";
                    let x = 0;

⤞ const

Variables defined with let can not be redeclare or let can not be reinitialize.

⤞ Syntax 
                        const age = 30;
                        const name = "John";

⤞ Types Of variables

  • 1. local
  • 2. global

⤞ Local

which variable define inside the function body or block

Local variable can only use inside that function only

⤞ Syntax 
                        function localVar() 
                        {
                            //declare the local variable
                            var local = 10;
                            console.log(local);
                        }
                        localVar();

⤞ Global

Which was define outside the function body.

We can use it any were in that file

⤞ Syntax 
                        //declare the global variable
                        var global = 10;
                        function globalVar() 
                        {
                            console.log(global);
                        }
                        globalVar();

⤞ Types Of value

  • 1. static
  • 2. dynamic

⤞ static

Which was define in the compile time(source code)

⤞ Syntax 
                        const x = 10; // Static value
                        const y = 1000; // Static value
                        const pi = 3.14159; // Static value
