Internal Javascript

➜   There are two method to embedded Javascript in webpage.
  • 1.   in head tag
  • 2.   in body tag

✧ in head tag

➜   JavaScript function is placed in the <head> section of an HTML page.

⤞Syntax

                    <!DOCTYPE html>
                    <html>
                        <head>
                            //internal JavaScript
                            <script>  </script>
                        </head>

                        <body>                   
                        //write the code

                        </body>
                    </html>

⤞Example

                        <!DOCTYPE html>
                        <html>
                        <head>
    
                        //internal JavaScript
                        <script>
                            function myFunction() {
                                document.getElementById("demo").innerHTML = "Paragraph changed.";
                            }
                        </script>
                        </head>
                        <body>
                        <h2>Demo JavaScript in Head</h2>
    
                        <p id="demo">A Paragraph</p>
                        <button type="button" onclick="myFunction()">Try it</button>
    
                        </body>
                        </html>


⤞Output

A Paragraph.
