Website

=> website is collection of web pages.

=> A website is a collection of related web pages, multimedia content, and other resources that are typically identified by a common domain name and are published on at least one web server.

There are two types of website

1. static website

2. dynamic website

✦ static website

=> Static websites are built using only HTML (Hypertext Markup Language) and CSS (Cascading Style Sheets).

=> The content remains fixed and does not change unless manually updated by a web developer.

✦ dynamic website

=> Dynamic websites use server-side scripting languages like PHP, Python, Ruby, or JavaScript (Node.js) to generate content.

=> Content is often stored in databases, allowing for easy updates and management.
