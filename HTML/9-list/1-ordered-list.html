◉Ordered list
→ Ordered list use when we arranged list in accesnding or descending order manner.
→ An ordered list starts with the <ol>tag.
→ syntex :- <ol> </ol>
→ Each list item starts with the <li> tag.
→ syntex :- <li> </li><li> </li>
→ example of Ordered list:
syntex:-
<ol>
     <li>Coffee<li>
     <li>Tea<li>
     <li>Milk<li>
</ol>
output:
  1. Coffee
  2. Tea
  3. Milk
