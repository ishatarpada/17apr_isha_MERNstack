◉Unordered list
→ Unordered list use when we don't want to arranged list in accesnding or descending order manner.
→ An Unordered list starts with the <ul>tag.
→ syntex :- <ul> </ul>
→ Each list item starts with the <li> tag.
→ syntex :- <li> </li>
→ example of Unordered list:
syntex:-
<ul>
     <li>Coffee<li>
     <li>Tea<li>
     <li>Milk<li>
</ul>
  • Coffee
  • Tea
  • Milk
 ▶ Attribute in Unordered list.
syntex :- <ul > <li>...</li> </ul>
