flex Property

 ❱ The flex property sets the flexible length on flexible items.

Flex Property
No name syntex Discription
1 flex-direction flex-direction : row | row-reverse | column | column-reverse The flex-direction property specifies the direction of the flexible items.
2 flex-wrap flex-wrap : wrap | no-wrap | wrap-reverse ; The flex-wrap property specifies whether the flexible items should wrap or not.
3 flex-shrink flex-shrink : number Displays an element as a block element (like <p>). It starts on a new line, and take full width
4 Justify-content Justify-content : flex-start|flex-end|center|space-between|space-around|space-evenly The justify-content property aligns the flexible container's items when the items do not use all available space on the main-axis (horizontally).
