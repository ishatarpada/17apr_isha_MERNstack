Display Property

 ❱ The display property specifies the display behavior (the type of rendering box) of an element.

☣   Display Syntex

❖   display: none | inline | block | inline-block | flex;
Display Property
No Value Discription Example
1 none The element is completely removed display : none
2 inline Displays an element as an inline element (like <span>). Any height and width have no effect display : inline
3 block Displays an element as a block element (like <p>). It starts on a new line, and take full width display : block
3 inline-block Displays an element as an inline-level block container. The element itself is formatted as an inline element, but you can apply height and width values display : inline-block
3 flex Displays an element as a block-level flex container display : flex
