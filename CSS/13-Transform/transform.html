transform Property

 ❱ The transform property applies a 2D or 3D transformation to an element. This property allows you to rotate, scale, move, skew, etc., elements.

☣   transform Syntax

❖   transform: skew() | rotate() | translate() | scale();
transform Property
No Property Name Description Example
1 skew() Defines a 2D skew transformation along the X- and the Y-axis skew property
2 translate Defines a 2D translation translate property
3 rotate() Defines a 2D rotation, the angle is specified in the parameter rotate property
4 scale() Defines a 2D scale transformation. scale Property
