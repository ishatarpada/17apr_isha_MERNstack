Outline-Style

⇥   The outline-style property specifies what kind of outline to display.
⇥   The outline-style property specifies the style of the outline, and can have one of the following values:

    ⇥  The following values are allowed:

  • dotted - Defines a dotted outline

    A dotted outline.

  • dashed - Defines a dashed outline

    A dashed outline.

  • solid - Defines a solid outline

    A solid outline.

  • double - Defines a double outline

    A double outline.

  • groove - Defines a 3D grooved outline. The effect depends on the outline-color value

    A groove outline.

  • ridge - Defines a 3D ridged outline. The effect depends on the outline-color value

    A ridge outline.

  • inset - Defines a 3D inset outline. The effect depends on the border-color value

    An inset outline.

  • outset - Defines a 3D outset outline. The effect depends on the outline-color value

    An outset outline.

  • none - Defines no outline

    No outline.

  • hidden - Defines a hidden outline

    A hidden outline.
