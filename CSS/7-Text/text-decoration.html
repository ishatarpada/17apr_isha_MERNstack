Text-Decoration

    ⇥ Text-decoration:-
  • text-decoration-line : Specifies the kind of text decoration to be used (underline, overline, etc.)
  • text-decoration-color : Specifies the color of the text-decoration
  • text-decoration-style : Specifies the style of the text decoration (solid, dotted, etc.)
  • text-decoration-thickness : Specifies the thickness of the text decoration line
  • text-decoration : Sets all the text-decoration properties in one declaration

text-decoration-line

⇥ The text-decoration-line property is used to add a decoration line to text.
⇥ You can combine more than one value, like overline and underline to display lines both over and under a text.
⇥ Example:-

⇥ Overline text decoration

⇥ Line-through text decoration

⇥ Underline text decoration

⇥ Overline and underline text decoration.

text-decoration-color

⇥ The text-decoration-color property is used to set the color of the decoration line.
⇥ Example:-

⇥ Overline text decoration with color

⇥ Line-through text decoration with color

⇥ Underline text decoration with color

⇥ Overline and underline text decoration with color

text-decoration-style

⇥ The text-decoration-style property is used to set the style of the decoration line.
⇥ Example:-

⇥ Underline text decoration with wavy style & color

⇥ Underline text decoration with double style & color

⇥ Underline text decoration with dotted style & color

⇥ Underline text decoration with dashed style & color

text-decoration-thickness

⇥ The text-decoration-thickness property is used to set the thickness of the decoration line.
⇥ Example:-

⇥ Underline text decoration double style & color &thickness

⇥ Underline text decoration wavy style & color &thickness

⇥ Underline text decoration dotted style & color &thickness

⇥ Underline text decoration dashed style & color &thickness
