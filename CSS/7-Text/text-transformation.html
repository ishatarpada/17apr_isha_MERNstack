Text Transformation

⇥   The text-transform property controls the capitalization of text.

    ⇥ Text-Transformation property value:-

  • text-transform: uppercase; 
    😊 helloo Namstee!....
  • text-transform: lowercase; 
    😊 helloo Namstee!....
  • text-transform: capitalize; 
    😊 helloo Namstee!....
