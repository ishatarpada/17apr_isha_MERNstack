border-image-width

⇥  The border-image-width property specifies the width of the border image.

  Syntex:- border-image-slice: number|%|fill ;

  • number :- Default value 1. Represents multiples of the corresponding border-width
  • % :- Refers to the size of the border image area: the width of the area for horizontal offsets, the height for vertical offsets
  • length :- Causes the middle part of the image to be displayed
border-image-width: 10px;

border-image-width: 10px 40px;

border-image-width: 10%;

border-image-width: 15% 50%;

border-image-width: 2;
