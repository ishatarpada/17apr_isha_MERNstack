Filter Property : Saturate

✒ Saturates the image.
✒ 0% (0) will make the image completely un-saturated.
✒ 100% is default and represents the original image.
✒ Values over 100% provides super-saturated results.
✒ Negative values are not allowed.

❋Example:-

filter: saturate(0%);


filter: saturate(25%);


filter: saturate(50%);


filter: saturate(75%);


filter: saturate(100%);
