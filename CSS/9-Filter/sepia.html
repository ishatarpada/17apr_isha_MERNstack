Filter Property : sepia

✒ Converts the image to sepia.
✒ 0% (0) is default and represents the original image.
✒ 100% will make the image completely sepia.
✒ Negative values are not allowed.

❋Example:-

filter: sepia(0%);


filter: sepia(25%);


filter: sepia(50%);


filter: sepia(75%);


filter: sepia(100%);
