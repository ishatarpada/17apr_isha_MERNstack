Filter Property : Invert

✒ Adjusts the contrast of the image.
✒ 0% (0) is default and represents the original image.
✒ 100% will make the image completely inverted.
✒ Negative values are not allowed.

❋Example:-

filter: invert(0%);

filter: invert(25%);

filter: invert(50%);

filter: invert(75%);

filter: invert(100%);
