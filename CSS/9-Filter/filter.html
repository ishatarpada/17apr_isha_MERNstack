Filter Property

 ❱ The filter property defines visual effects (like blur and saturation) to an element (often ).

☣   Filter Syntex

❖   filter: none | blur() | brightness() | contrast() | drop-shadow() | grayscale() | hue-rotate() | invert() | opacity() | saturate() | sepia() | url();
Filter Property
No Property Name Discription Example
1 blur(px) Applies a blur effect to the image. Blur property
2 brightness(%) Adjusts the brightness of the image. Brightness property
3 contrast(%) Adjusts the contrast of the image. Adjusts property
4 grayscale(%) Converts the image to grayscale. Grayscale Property
5 Invert Inverts the samples in the image. Invert Property
6 opacity(%) Sets the opacity level for the image. The opacity-level describes the transparency-level. opacity Property
7 Saturates Saturates the image. Saturate Property
8 sepia Converts the image to sepia. sepia Property
9 drop-shadow Applies a drop shadow effect to the image. drop-shadow Property
10 hue-rotate(deg) Applies a hue rotation on the image.
The value defines the number of degrees around the color circle the image samples will be adjusted.
0deg is default, and represents the original image.		 hue-rotate Property
