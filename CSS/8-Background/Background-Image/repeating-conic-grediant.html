Background Image - Repeating Conic Grediant

⇥ The repeating-conic-gradient() function is used to repeat conic gradients.

⇥   SYNTEX:-
   ⥁ background-image: repeating-conic-gradient([from angle] [at position,] color degree, color degree, ...);

  • from angle
    ⇴ Optional. The entire conic gradient is rotated by this angle. Default value is 0deg

  • color degree, ..., color degree
    ⇴ Color stops are the colors you want to render smooth transitions among. This value consists of a color value, followed by an optional stop position (a percentage between 0% and 100% or a length along the gradient axis).

  • at position
    ⇴ Defines the position of the gradient. Default is "center"Optional. Specifies the gradient center of the conic gradient. Default value is center

☻ Example:-

 ⇰ background-image: repeating-conic-gradient(rgb(229, 168, 231) 10%, rgb(50,2,52 )20%);


☻ A repeating conic gradient with defined color-starts and color-stops:

 ⇰ background-image: repeating-conic-gradient(rgb(229, 168, 231) 10%, rgb(50,2,52 )20%);
