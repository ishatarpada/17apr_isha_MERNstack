Background Image - conic-grediant

↬  The conic-gradient() function sets a conic gradient as the background image.
↬  A conic gradient is a gradient with color transitions rotated around a center point.
↬  To create a conic gradient you must define at least two color stops.

⇥   SYNTEX:-
   ⥁ background-image: repeating-conic-gradient([from angle] [at position,] color degree, color degree, ...);

  • from angle
    ⇴ Optional. The entire conic gradient is rotated by this angle. Default value is 0deg

  • color degree, ..., color degree
    ⇴ Color stops are the colors you want to render smooth transitions among. This value consists of a color value, followed by an optional stop position (a percentage between 0% and 100% or a length along the gradient axis).

  • at position
    ⇴ Defines the position of the gradient. Default is "center"Optional. Specifies the gradient center of the conic gradient. Default value is center
⇥ example:-

    ⇥ background-image: conic-gradient(teal, rgb(8, 57, 57), rgb(18, 57, 57));

⇥ A conic gradient with five colors:

    ⇥ background-image: conic-gradient(red, yellow, green, blue, black);

⇥ A conic gradient with three colors and a degree for each color:

    ⇥ background-image: conic-gradient(red 45deg, yellow 90deg, green 210deg)

⇥ Make the conic gradient look like a pie by adding border-radius: 50%:

    ⇥ background-image: conic-gradient(red, yellow, green, blue, black);

⇥ A conic gradient with a from angle:

    ⇥ background-image: conic-gradient(from 90deg, red, yellow, green);

⇥ A conic gradient with an at position:

    ⇥ background-image: conic-gradient(at 60% 45%, red, yellow, green);

⇥ A conic gradient with both from angle and at position:

    ⇥ background-image: conic-gradient(from 90deg at 60% 45%, red, yellow, green);

⇥ Another pie chart:

    ⇥ background-image: conic-gradient(red 0deg, red 90deg, yellow 90deg, yellow 180deg, green 180deg);
