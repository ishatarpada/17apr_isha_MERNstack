Question-5 What the deference between undefined and undeclare in JavaScript?

No undefined undeclared
1 An undeclared variable is a variable that has been used in the code without being declared using the var, let, or const keyword. If we try to use a variable without declaring it, JavaScript will create a global variable or conduct the Error
2 An undefined variable is a variable that has been declared but has not been assigned a value or has been explicitly set to the undefined value. When you declare a variable using var, let, or const but don't assign a value to it, the variable is initialized with the value undefined.
