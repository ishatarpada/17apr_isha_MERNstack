Question-1 What is JavaScript. How to use it?

=> JavaScript is the client side scripting language

=> JavaScript is a high-level, interpreted programming language primarily used for client-side web development.

=> It allows developers to add interactive elements, dynamic behavior, and functionality to websites.

To use JavaScript:

  1. In Web Development:

    => Inline JavaScript: You can include JavaScript code directly within HTML using <script> tags..

  2. External JavaScript File:

    => create a separate JavaScript file (with a .js extension) and link it to your HTML file using <script> tags.
