What is === operator?

➛ The === operator in JavaScript is the strict equality operator.
➛ It is used to compare two values for equality without performing type coercion.
➛ it checks both the values and their types, and the comparison will only true if both the values are of the same type and have the same content.

Example:-

                document.write(5 === 5);     
                document.write(5 === '5');     
                document.write(5 === 6);
