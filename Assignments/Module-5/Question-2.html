How to Apply CSS Using JQuery, How to Add Class and Remove Class in Jquery, JQuery Animation? ?

Apply CSS using jQuery :
➛ css() method is use to apply css using jQuery.
➛ This method allows you to get or set the value of one or more CSS properties for a selected element
➛ jQuery also simplifies a lot of the complicated things from JavaScript, like AJAX calls and DOM manipulation.

jQuery code
⚛ Set a Single CSS Property:
            $(document).ready(function () {
                $("#btn").click(function () {
                    $('#content').css('color', 'green');
                });
            });

output

⚛ Set Multiple CSS Properties:
            $(document).ready(function () {
                $("#btn1").click(function () {
                    $('#content1').css({
                        'color': 'darkslategrey',
                        'font-size': '35px'
                    });
                });
            });


output
Hello ishaaaaa
Adding or Modifying Classes:
jQuery code
⚛ adding a Class :- Adds one or more classes to the selected elements
            $(document).ready(function () {
                $("#btn2").click(function () {
                    $("h3, h4").addClass("color");
                    $("h4").addClass("size");
                });
            });

output

Hello

isha




⚛ removing a Class :- Removes one or more classes from the selected elements
            $(document).ready(function(){
                $("#btn3").click(function(){
                  $("#w-1").remove();
                  $("#w-2").remove();
                });
              });

output

Hello

isha




⚛ Toggling a Class :- Toggles between adding/removing classes from the selected elements
            $(document).ready(function () {
                $("#btn4").click(function () {
                    $("b, i").toggleClass("toggle");
                });
            });

output
Hello isha



Animation:

➛ The jQuery animate() method is used to create custom animations.

jQuery code 
            $(document).ready(function(){
                $("button").click(function(){
                  $("div").animate({left: '250px'});
                });
              });
