What is the use of isNaN function ?

➛ The isNaN function in JavaScript is used to checked whether a value is number or not.
➛ The isNaN function takes a single argument and returns true if the argument is NaN, and false otherwise.
➛ It can be used to check whether a variable or a value is a valid number.

Example:-

                document.write(Number.isNaN(123));      
                document.write(Number.isNaN('Hello'));  
                document.write(Number.isNaN('123'));    
                document.write(Number.isNaN(NaN));      
                document.write(Number.isNaN(1 / 0));
