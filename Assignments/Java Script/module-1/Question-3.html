Question-3 Define a Data Types in js?

There are two type of datatype in JavaScript

  1. Primitive
    1. Number

      => Represents numeric values, including integers and floating-point numbers.

    2. String

      => Represents textual data enclosed in single ('') or double ("") quotes.

    3. Boolean

      => Represents logical values true or false.

    4. Undefined

      => Represents a variable that has been declared but not assigned a value.

    5. Null

      => Represents the intentional absence of any value or an empty value.

    6. Symbol

      => Represents a unique and immutable value used as an identifier for object properties.

  2. Non-primitive
    1. Object

      =>Represents a collection of properties (key-value pairs).

    2. Function

      => Represents a reusable block of code that can be called by name.

    3. Array

      => Represents a collection of elements stored in a specific order.

    4. Date

      =>Represents a date and time value.

    5. RegExp

      => Represents a regular expression pattern used for pattern matching in strings.
