Question-2 How many type of Variable in JavaScript?

There are three type of variable in JavaScript

  1. Global Scope

    => Variables declared outside of any function or block

    => Global variables are accessible from anywhere in the code

  2. Local Scope

    => Variables declared inside of any function or block

    => Global variables are not accessible from anywhere in the code only accessible within the function where they are defined

  3. Block Scope

    => Variables declared using let and const have block scope

    => Block scope means the variable is only accessible within the block where it's defined, such as within curly braces {} in if statements, loops, and function blocks.

There are three way to declared variable in JavaScript

  1. var

    => The var keyword is the oldest way to declare variables in JavaScript.

    => Variables declared with var have function scope or global scope (if declared outside any function).

  2. let

    => The let keyword was introduced in ES6 and provides block-level scoping.

    => let keyword cannot be redeclare within the same scope, but their values can be changed.

  3. const

    => The const keyword also came with ES6 and is used to declare constants

    => const keyword cannot be redeclare or value can't reassigned.
