How can the style/class of an element be changed?

Syntax:-
HTML file 
            <!DOCTYPE html>
            <html lang="en">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>Question-8</title>
                <link rel="shortcut icon" href="../../image/favikon.jpg" type="image/x-icon">
                <link rel="stylesheet" href="Question-8.css">

            </head>
            <body>

                <div id="content">  Hello ishaaaaa </div>
                
                <button onclick="changeColor()">change color</button>

                //path give the js file
                <script src="script/Question-8.js"></script>
            </body>


JavaScript file 
            function changeColor() {

                var content = document.getElementById("content");
                content.style.color = '#3c0932';
                content.style.fontSize = '40px';
            }
