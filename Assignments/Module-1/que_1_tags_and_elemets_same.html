Question-1.  Are the HTML tags and elements the same thing?

Answer :-
⇒ HTML tags and elements are not same.

⇒ beacuase tags means use to tell the browser how to show the content or information on web browser.

⇒ for example :- <title></title> , <b></b> , <br>

➽ There are two type of tag .
1. pair tag
→ for example:<title></title>
2.Non Pair / Void /self / empty Tag.
→ for example:- <br> ,<hr> ,<link>

⇒ elements means combination of starting tag and closing tag and content.

⇒ for example :- <title>Question-1 </title> , <b>Lorem, ipsum dolor sit amet consectetur adipisicing elit. Assumenda, maiores.</b>
