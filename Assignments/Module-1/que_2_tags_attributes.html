Question 2.   What are tags and attributes in HTML?

Answer:-

➽Tags
→ Tags means use to tell the browser how to show the content or information on web browser.

→ for example :- <title></title> , <b></b> , <br>

⇒ There are two type of tag .

1.pair tag

→pair tag means which tag they have starting tag and closing tag.

→ for example:<title></title>

2.Non Pair / Void /self / empty Tag.

→Non pair tag which tag they have only starting tag.

→ for example:- <br> ,<hr> ,<link>

➽Attributes

Attributes means define the property of the element.

→Attribute is use to define additional information

→Attribute is compulsory define inside the start tag

→ for example:- <body bgcolor="lightgray"> </body>
