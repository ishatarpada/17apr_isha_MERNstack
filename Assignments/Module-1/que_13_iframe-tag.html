Question 13   What is the use of an iframe tag?

⟹ The <iframe> tag specifies an inline frame.
⟹ The <iframe> tag in HTML is used to embed another HTML document within the current document.
⟹ It allows you to display content from another source, such as a website or web page, within the context of your own webpage.


Embed path

syntex:-  <iframe src="url" title="description"></iframe>
example:-  <iframe src="../../HTML/7_image-tag/3_image-slider.html" height="450" width= 100% title="Iframe Example"></iframe>

     

Embed website

syntex:-  <iframe src="url" title="description"></iframe>
example:-  <iframe src="https://www.amazon.com/" height="450" width= 100% title="Amazon"></iframe>
