Question-17 What are the different tags to separate sections of text?

😀  Tags of Separate Section

<br>
→ <br> tag is used to separate the line of text.
→ It breaks the current line and conveys the flow to the next line
<p>
→ The <p> tag contains the text in the form of a new paragraph.
<blockquote>
→ It is used to define a large quoted section.
→ If you have a large quotation, then put the entire text within <blockquote>.............</blockquote> tag.
<div>
→ The <div> tag defines a division or a section in an HTML document..
→ The <div> tag is used as a container for HTML elements - which is then styled with CSS or manipulated with JavaScript.
<h1> to <h6>
→ It is used to define a heading section.
→ The <h1> to <h6> tags are used to define HTML headings.
