◉ The Target Attribute
→ The target attribute specifies where to open the linked document.
→ The target attribute can have one of the following values:
   1.) _self - Default. Opens the document in the same window/tab as it was clicked
   syntex : <a  href="url"  target=" ">link text</a>
  → for example:- filpkart
   2.) _blank - Opens the document in a new window or tab
   syntex : <a  href="url"  target="_blank">link text</a>
  → for example:- Myntra
   3.) _parent - Opens the document in the parent frame
   syntex : <a  href="url"  target="_parent">link text</a>
  → for example:- Google
   4.) _top - Opens the document in the full body of the window
   syntex : <a href="url"  target="top">link text</a>
  → for example:- Amazon
