Question 11   What is the difference between a block-level element and an inline element?

Inline Element and Block-level Element

no. Inline element Block-level element
1 Inline elements takes up as much width as necessary or required. Block-level element always takes up the full width available.
2 Inline element doesn't start in new line. Block-level element start in new line and the browsers automatically add some space before and after the element.
3 Inline elements allow other inline elements to sit behind. Block-level elements doesn't allow other elements to sit behind.
4 Inline elements don't have top and bottom margin. Block elements have top and bottom margin.
5 example of inline element is: <b>...</b> , <u>...</u> , <i>...</i> , <strong>...</strong> example of block-level element is: <p>...</p> , <br>...</br> , <li>...</li> , <blockquote>...</blockquote>
