Question 10   How can we club two or more rows or columns into a single row or column in an HTML table?

➜ we merge multiple rows or columns into a single row or column in a table using the "rowspan" and "colspan" attributes.
➜ To merge rows, we can use the rowspan attribute on a or element to specify the number of rows the cell should span.
➜ For example, if we want to merge two rows into one, you would set rowspan="2" on the cell you want to span multiple rows.
➜ To merge columns, we can use the rowspan attribute on a or element to specify the number of columns the cell should span.
➜ For example, if we want to merge two columns into one, you would set colspan="2" on the cell you want to span multiple columns.
