What is the rule set?

✒ The rule set means how to declare style in html documents and specific HTML element or group of elements should be styled on a web page or websites.
✒ A rule set consists of a selector and a declaration block.

❖ Selector

✒ The selector is used to target HTML elements to which the styles should be applied.
✒ there are types of Selector.

    Types of selectors
  • ✦ Universal Selector
  • ✦ id-selector
  • ✦ class Selector
  • ✦ Group Selector
  • ✦ child Selector
  • ✦ general-sibling Selector
  • ✦ adjust-sibling Selector
  • ✦ element Selector
  • ✦ pseudo-class Selector
  • ✦ pseudo-element Selector

❖ Universal Selector

⇒ The universal selector (*) selects all HTML elements on the page.

Syntax 
                        *{
                            font-family:  Georgia, serif;
                            font-weight: 500;
                        }

❖ id Selector

⇒ The id selector uses the id attribute of an HTML element to select a specific element.
⇒ To select an element with a specific id, write a hash (#) character, followed by the id of the element.
⇒  here in this example , id name is syntax.

Syntax 
                        #syntax{
                            color: #510651;
                            font-family : Georgia, serif;
                            font-weight: 600;
                        }

❖ class Selector

⇒   The class selector selects HTML elements with a specific class attribute.
⇒   To select elements with a specific class, write a period (.) character, followed by the class name.
⇒   here in this example , class name is selector.

Syntax 
                        .selector {
                            font-family: Georgia, serif;
                        }

❖ declaration block

⇒   The declaration block is enclosed in curly braces {} and contains one or more property-value pairs.

Syntax 
                        {
                            color: blue;
                            font-size: 16px;
                            margin: 10px;
                          }
