Question-13 What are the ways to integrate CSS as a web page?

☣   There are three ways to integrate CSS as a web page.

  1. Inline CSS
  2. Internal CSS
  3. External CSS
➩ inline css
 ➜ An inline style used to apply style for a single element.
Syntax:-
                    <h5 style="background-color="navy">Syntax</h5>

➩internal css
 ➜ An internal style sheet used one single HTML page has a unique style.
 ➜ The internal style is defined the <style> element, inside the head section.
Syntax:-
                    <style>
                        .container{
                            height: auto;
                            width: 100%;
                        }
                    </style>

➩external css
 ➜ external style sheet used for the change the look of an entire website by using just one file.
 ➜ Each HTML page must include a reference to the external style sheet file inside the <link> element, inside the head section.
Syntax:-
                    <link rel="stylesheet" href="css file-name">
