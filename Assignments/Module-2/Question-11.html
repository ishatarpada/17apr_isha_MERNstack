How to center block elements using CSS1?

✒ the center tag <center> </center> is used for center the block elements.

Example:-

                <!DOCTYPE html>
                <html lang="en">

                <head>
                    <meta charset="UTF-8">
                    <meta name="viewport" content="width=device-width, initial-scale=1.0">
                    <title>Question-11</title>
                    <link rel="shortcut icon" href="../../image/favikon.jpg" type="image/x-icon">
                
                    <style>   
                    //internal css for center the box
                    .container .box{
                        height: 500px;
                        width: 500px;
                        background: linear-gradient(to right , lightseagreen , teal);
                        border: 2px ridge burlywood;
                        position: relative;
                        overflow: hidden;
                    }

                    .container .box .boxs{
                        height: 500px;
                        width: 500px;
                        transition: 5s;
                        background-color: #00808080;
                    }

                    .container .box .box-1{
                        position: absolute;
                        top: -500px;
                        left: -500px;
                    }

                    .container .box .box-2{
                        position: absolute;
                        top: 500px;
                        left: 500px;
                    }

                    .container .box .box-3{
                        position: absolute;
                        top: 500px;
                        left: -500px;
                    }

                    .container .box .box-4{
                        position: absolute;
                        top: -500px;
                        left: 500px;
                    }

                    .container .box:hover .boxs{
                        position: absolute;
                        top: 0;
                        left: 0;
                    }

                <style>
                <head>
                
                <body>
                <center> tag is used for center the block elements
                        <center>
                            <div class="box">
                                <div class="box-1 boxs"> </div>
                                <div class="box-2 boxs"> </div>
                                <div class="box-3 boxs"> </div>
                                <div class="box-4 boxs"> </div>
                            </div>
                        </center>
                </body>

                </html>
