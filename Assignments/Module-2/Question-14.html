Question-14 What is embedded style sheets?

➜   embedded style sheets is the internal css.
➜   An internal style sheet used one single HTML page has a unique style.
➜   The internal style is defined the <style> element, inside the head section.

Syntax:-
                <!DOCTYPE html>
                <html lang="en">

                <head>
                    <meta charset="UTF-8">
                    <meta name="viewport" content="width=device-width, initial-scale=1.0">
                    <title>Question-14</title>
                    <link rel="shortcut icon" href="../../image/favikon.jpg" type="image/x-icon">

                    //internal css //
                    <style>
                        .div {
                            text-align: left;
                            font-family: 'Times New Roman', Times, serif;
                            padding: 2px;
                            background-color: #89b5b5;
                            width: 50%;
                            color: white;
                        }
                    </style>
                    
                </head>
