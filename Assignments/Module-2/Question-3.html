Question-3 What is the difference between CSS2 and CSS3 ?

no. CSS2 CSS3
1 Basic selectors like class, ID, tag selectors Advanced selectors like pseudo class and pseudo elements
2 In CSS2 , Limited layout options. In CSS3 , Many layout options like flex grid and responsive etc.
3 basic color properties are available in CSS2. There are many Properties are available like rgb , rgba , hsl , hsla etc.
4 basic background-color properties are available in CSS2. There are many Properties are available like background-image or gradient property like linear gradient and redial gradient etc.
5 Transitions or animations properties are not available in CSS2 Transitions or animations properties are available in CSS3
6 media queries are not supported in CSS2 Responsive design with media queries is available in CSS3
7 Limited font choices and text style in CSS2 many font style and text style like text-shadow in CSS3
