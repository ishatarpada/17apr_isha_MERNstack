What are the media types allowed by CSS?

=> there are some media types allowed by CSS.
    1.all
    2.screen
    3.speech
    4.print

all

This is the default media type, and it applies to all devices.

example:-
                    @media all {
                        /* CSS rules specific to the media type */
                        body{
                            font-size: 16px;
                            background-color: lightgreen;
                        }
                    }

screen

The default media type for styles intended for computer screens, tablets, and similar devices.
This is the most commonly used media type for web design.

example:-
                    @media screen and (min-width: 480px) {
                       /* CSS rules specific to the media type */
                        body {
                          background-color: lightgreen;
                        }
                      }

specch

Used for screen readers and other speech synthesis devices.

example:-
                    @media speech and (min-width: 480px) {
                        /* CSS rules specific to the media type */
                        body {
                            background-color: lightgreen;
                          }
                      }

print

Used for styles intended for print media. Styles defined for this media type will be applied when the user prints the web page.

example:-
                    @media print and (min-width: 480px) {
                        /* CSS rules specific to the media type */
                        body {
                          background-color: lightgreen;
                        }
                      }
