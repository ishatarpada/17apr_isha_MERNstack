What are the external style sheets?

➩ external css
  ➜   external style sheet used for the change the look of an entire website by using just one file.
  ➜   Each HTML page must include a reference to the external style sheet file inside the <link> element, inside the head section.

Syntax:-
HTML file 
                <!DOCTYPE html>
                <html lang="en">
                <head>
                    <meta charset="UTF-8">
                    <meta name="viewport" content="width=device-width, initial-scale=1.0">
                    <title>Question-15</title>
                    <link rel="shortcut icon" href="../../image/favikon.jpg" type="image/x-icon">

                    
                    //path give the css file (external css)//
                    //<link rel="stylesheet" href="css file-name">//
                    
                    <link rel="stylesheet" href="Question-15.css">

                </head>


CSS file 
                .container{
                    height: auto;
                    width: 100%;
                }
