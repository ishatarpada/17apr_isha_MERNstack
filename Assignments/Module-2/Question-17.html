Question-17 What is the meaning of the CSS selector?

☣ Selectors

⇒CSS selectors are used to select the HTML elements you want to style.
⇒There are many selector available in CSS , like id selector , class selector , pseudo-class selector , pseudo-element selector.

  • ✦ Universal Selector
  • ✦ id-selector
  • ✦ class Selector
  • ✦ Group Selector
  • ✦ child Selector
  • ✦ general-sibling Selector
  • ✦ adjust-sibling Selector
  • ✦ element Selector
  • ✦ pseudo-class Selector

❖ Universal Selector

⇒ The universal selector (*) selects all HTML elements on the page.

Syntax 
                        *{
                            font-family:  Georgia, serif;
                            font-weight: 500;
                        }

❖ id Selector

⇒ The id selector uses the id attribute of an HTML element to select a specific element.
⇒ To select an element with a specific id, write a hash (#) character, followed by the id of the element.
⇒   here in this example , id name is syntax.

Syntax 
                        #syntax{
                            color: #510651;
                            font-family : Georgia, serif;
                            font-weight: 600;
                        }

❖ class Selector

⇒   The class selector selects HTML elements with a specific class attribute.
⇒   To select elements with a specific class, write a period (.) character, followed by the class name.
⇒   here in this example , class name is selector.

Syntax 
                        .selector {
                            font-family: Georgia, serif;
                        }

❖ group Selector

⇒   The grouping selector selects all the HTML elements with the same style.
⇒   This reduces the code and extra effort to declare common styles for each element.
⇒   To group selectors, each selector is separated by a space( ) or comma ",".
⇒   here in this example , h1 and p are select using group selector.

Syntax 
                        h1 , p{    
                            margin: auto;
                            width: 100%;
                            padding: 2px;
                            font-family: Georgia, serif;
                        }

❖ child Selector

⇒   The child combinator (>) is placed between two CSS selectors.
⇒   The child selector selects all elements that are the children of a specified element.
⇒   if <div> is the parent tag and child tag is <h1> , than <h1> select <div>   >  <h1>

Syntax 
                        div > p{
                            height: 100px;
                            width: 75%;
                            margin: auto;
                            color: rgb(70, 7, 41);
                            font-size: 25px;
                        }

❖ adjust-sibling Selector

⇒   The adjacent sibling selector is used to select an element that is directly after another specific element.
⇒   The following example selects the first <p> element that are placed immediately after <div> elements:

Syntax 
                        div + p {
                            padding: 5px;
                            margin : 5px;
                            width: fit-content;
                           font-size: 25px;
                        }


❋ pseudo-class

The Pseudo-class is selected : this sign.

❖ active Selector

⇒   The :active selector is used to select and style the active link.
⇒   A link becomes active when you click on it.
⇒   in this example , div is selected by the active selector

Syntax 
                        div:active{
                            height: 300px;
                            font-size: 20px;
                            font-weight: 900;
                        }

❖ hover Selector

⇒   The :hover selector is used to select elements when you mouse over them.
⇒   in this example , div is select when mouse over them using the hover selector

Syntax 
                        div:hover{
                            height: 300px;
                            font-size: 20px;
                        }

❖ focus Selector

⇒   The :focus selector is used to select the element that has focus.

Syntax 
                        input:focus{
                            outline: none; 
                            background-color: #305653;
                            color: white;
                        }

❖ nth-child Selector

⇒   The :nth-child(n) selector matches every element that is the nth child of its parent.
⇒   n can be a number, a keyword (odd or even), or a formula (like an + b).

Syntax 
                        .child :nth-child(even){
                            background-color: #396565;
                            margin: 2px;
                        }
                        
                        .child :nth-child(odd){
                            background-color: #79a294;
                            margin: 2px;
                        
                        }


❋ pseudo-element

The Pseudo-element is selected :: this sign.

❖ after Selector

⇒  The ::after selector inserts something after the content of each selected element(s).
⇒  Use the content property to specify the content to insert.
⇒   in this example , content insert after p tag using after selector(p::after).

Syntax 
                        p::after{
                            content: url("helloo...");
                        }

❖ before Selector

⇒  The ::before pseudo-element can be used to insert some content before the content of an element.
⇒  Use the content property to specify the content to insert.
⇒   in this example , content insert before p tag using before selector(p::before).

Syntax 
                        p::before{
                            content: url("hellooo...");
                        }

❖ first-latter Selector

⇒ The ::first-line pseudo-element is used to add a special style to the first line of a text.
⇒ The ::first-line pseudo-element can only be applied to block-level elements.

Syntax 
                        p::first-letter{
                            color: #881168;
                            font-weight: 900;
                        }

❖ first-line Selector

⇒ The ::first-line pseudo-element is used to add a special style to the first line of a text.
⇒ The ::first-line pseudo-element can only be applied to block-level elements.

Syntax 
                        p::First-Line{
                            color: #12093a;
                            font-weight: 900;
                        }

marker Selector

⇒ The ::marker selector selects the marker of a list item.

Syntax 
                        ::marker{
                            color: #491d34;
                        }

❖ placeholder Selector

⇒ The ::placeholder selector selects form elements with placeholder text, and let you style the placeholder text.
⇒The placeholder text is set with the placeholder attribute, which specifies a hint that describes the expected value of an input field.

Syntax 
                    input:focus{
                        outline: none;
                        background-color: transparent;
                        border: none;
                        border-bottom: 2px solid lightblue;
                        color: rgb(15, 30, 57);
                    }

❖ section Selector

⇒The ::selection pseudo-element matches the portion of an element that is selected by a user.
⇒The following CSS properties can be applied to ::selection: color, background, cursor, and outline.
⇒The following example makes the selected text darkpink on a lightpink background:

Syntax 
                        p::selection{
                            color: #ead5e4;
                            background-color: #2e0a1d;
                            font-weight: 900;
                        }
