What are undeclared and undefined variables?

☣ Undeclared variable

➛ An undeclared variable is a variable that has been used in the code without being declared using the var, let, or const keyword.
➛ If you try to use a variable without declaring it, JavaScript will create a global variable or conduct the Error

Example:-

                undeclaredVar = 10;
                document.write(undeclaredVar);

☣ Undefined variable

➛ An undefined variable is a variable that has been declared but has not been assigned a value or has been explicitly set to the undefined value.
➛ When you declare a variable using var, let, or const but don't assign a value to it, the variable is initialized with the value undefined.

Example:-

                let undefinedVar;
                document.write(undefinedVar);
