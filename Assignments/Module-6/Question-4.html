What are the default Bootstrap text settings?

➺ Bootstrap uses a default font-size of 16px, and its line-height is 1.5.

➺ The default font-family is "Helvetica Neue", Helvetica, Arial, sans-serif.

✦ Text Alignment:

➺.text-start: Left-aligns the text.

➺ .text-center: Centers the text.

➺ .text-end: Right-aligns the text.

➺ .text-justify: Justifies the text.

✦ Text Transform:

➺.text-lowercase: Transforms the text to lowercase.

➺ .text-uppercase: Transforms the text to uppercase.

➺ .text-capitalize: Capitalizes the first letter of each word.
