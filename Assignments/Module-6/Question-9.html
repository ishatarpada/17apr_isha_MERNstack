In Bootstrap 4, what is flexbox

➺ In Bootstrap 4, the flexbox model is used to create flexible and responsive layouts.

➺ Flexbox is a layout system that allows you to design complex layouts more efficiently and with less code

➺ It is particularly useful for creating dynamic and resizable components within a container.

✦Grid System :

➺ The grid system in Bootstrap 4 is built on flex containers and flex items.

➺ Rows become flex containers, and columns become flex items.

➺ This makes it easy to create responsive and flexible layouts that adapt to different screen sizes.

✦Flex Container Classes:

➺ Bootstrap provides classes to turn an element into a flex container.

➺ The main class for this purpose is d-flex, which you can apply to a parent element to make its children flex items.

✦Alignment:

➺ Flexbox allows you to easily align items both horizontally and vertically.

➺ Bootstrap 4 provides classes like justify-content-* and align-items-* to control the alignment of flex items.
