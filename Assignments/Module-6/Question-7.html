What is a Button Group, and what is the class for a basic Button Group?

➺ Button Group is a group of buttons that are visually connected and often used to toggle between different options or actions.

➺ Button Group which is used to create a series of buttons in groups (without spaces) vertically or horizontally.

➺ In Bootstrap 5, the basic class for creating a Button Group is btn-group.

➺ There are also additional classes you can use to customize the appearance and behavior of the button group, such as btn-group-vertical to create a vertical button group.
