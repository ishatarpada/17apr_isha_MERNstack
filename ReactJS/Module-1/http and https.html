HTTP & HTTPS
no HTTP HTTPS
1 HTTP stands for HyperText Transfer Protocol. HTTPS stands for HyperText Transfer Protocol Secure.
2 In HTTP, the URL begins with “http://”. In HTTPS, the URL begins with “https://”.
3 HTTP uses port number 80 for communication. HTTPS uses port number 443 for communication.
4 HTTP Works at the Application Layer. HTTPS works at the Transport Layer.
5 HTTP does not use encryption, which results in low security in comparison to HTTPS. HTTPS uses Encryption which results in better security than HTTP.
6 HTTP speed is faster than HTTPS. HTTPS speed is slower than HTTP.
7 HTTP is used to transfer text, video, and images via web pages. HTTPS is used to transfer data securely via a network.
