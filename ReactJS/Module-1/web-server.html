Web Server

=> A web server is software used to serve web pages or transfer them to users.

=> A web server is a software application that handles incoming requests from web browsers and serves the appropriate web pages or resources to users over the internet.

=> When we enter a URL in your browser , our browser sends a request to the web server hosting the requested website. The web server then processes the request, retrieves the necessary files and sends them back to the user's browser, which produces the web page for display.

=> web servers include Apache, Nginx, Microsoft Internet Information Services (IIS), and others.
